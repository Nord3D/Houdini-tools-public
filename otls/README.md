
### Edge Group to Curve
  Давно хотел сделать замену "неспешному" лабсовскому Edge Group To Curve.
1. В моей версии убран лишний, как мне кажется, функционал (странный трансфер атрибутов, который можно сделать обычным ray sop'ом).
